# 1장

1. 시간복잡도 : 데이터 개수(N), 시간제한 1초
   1. N < 500 : O(N***3)
   2. N < 2000 : O(N**2)
   3. N < 100,000 : O(N logN )
   4. N < 10,000,000 : O(N)

2. 공간 복잡도 : 리스트 내의 데이터 개수는 1,000만 단위 이내.

   **공간복잡도는 대부분 python에서 고려 안해도됨**
